# Horizontal scroll

CRM drawer prototype for the horizontal status bubbles from Figma.

## Included states

- Drawer layout with the deal title, actions, pipeline selector, status bubbles, tabs, body, and footer.
- Prototype-only switcher buttons:
  - `–Ф—А–Њ–≤–µ—А –і–Њ–±–∞–≤–ї–µ–љ–Є—П`: `–Х—Б—В—М —Б–Ї—А–Њ–ї–ї`, `–љ–µ—В —Б–Ї—А–Њ–ї–∞`
  - `–Ф—А–Њ–≤–µ—А –і–Њ–±–∞–≤–ї–µ–љ–љ–Њ–≥–Њ`: `–Х—Б—В—М —Б–Ї—А–Њ–ї–ї`, `–љ–µ—В —Б–Ї—А–Њ–ї–∞`

The switcher buttons are intentionally visual only for now.

## Horizontal scroll interactions

- Left and right arrows scroll by the currently visible group of bubbles.
- Mouse wheel over the bubble row scrolls horizontally.
- Pointer drag on the bubble row scrolls relative to the drag distance.
